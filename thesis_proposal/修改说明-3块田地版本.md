## 📊 重新定义数据集

### 3块田地设置

| 田块 | 种植条件 | 特点 | 样本数 |
|------|---------|------|--------|
| 田块1 | 标准管理 | 正常种植密度，标准施肥 | 1200张 |
| 田块2 | 高密度种植 | 密度增加30%，氮肥偏高 | 1000张 |
| 田块3 | 低密度+控肥 | 密度降低20%，控氮增钾 | 800张 |

**总计**: ≥3000张RGB-D图像对

### 差异来源

虽然是同一地区，但3块田地仍有差异：

1. **种植条件差异**
   - 种植密度：影响冠层密度和光照穿透
   - 水肥管理：影响长势和叶色
   - 倒伏程度：不同管理导致倒伏率差异

2. **微环境差异**
   - 局部地形：田面平整度略有差异
   - 排水条件：影响土壤湿度
   - 遮挡情况：周边树木、建筑影响光照

3. **时相差异**
   - 不同时间采集（早中晚、阴晴天）
   - 不同生育期（分蘖、抽穗、灌浆、成熟）
   - 倒伏发展阶段（新鲜倒伏vs陈旧倒伏）

虽然只有3块田地，但可以这样包装：

**角度1: 数据集本身的价值**
```
✓ 首个水稻倒伏RGB-D数据集（填补空白）
✓ 高质量深度数据（LiDAR DSM，非普通RGB-D相机）
✓ 精细标注（像素级，包含倒伏程度）
✓ 多时相覆盖（完整生育期）
✓ 元数据丰富（GPS、环境参数、倒伏原因）
```

**角度2: 数据多样性**
```
虽然是3块相邻田地，但包含：
✓ 不同种植密度（低、中、高）
✓ 不同管理方式（标准、高氮、控氮）

✓ 倒伏发展阶段（新鲜、陈旧）

→ 数据内部多样性高
→ 适合研究外观不变性
```

**角度3: 实际应用导向**
```
✓ 模拟真实应用场景（同一地区多田块推广）
✓ 跨田块泛化（农户A的模型用于农户B）
✓ 实用价值高（符合推广需求）
```

## 🎯 论文逻辑调整

### 新的研究动机

```markdown
虽然3块田地位于同一地区，但由于：
1. 种植条件差异（密度、水肥）
2. 微环境差异（局部地形、排水）
3. 倒伏特征多样（程度、原因、时期）
4. 采集条件变化（时间、光照、角度）

导致外观特征存在显著差异，而倒伏的几何特征
（高度降低、角度倾斜）保持一致。

这为研究外观不变性学习提供了理想的测试场景。
```

### 跨域泛化的新定义

**不再是"跨地理位置"，而是"跨种植条件"**：

```
场景1: 跨种植密度泛化
       训练: 标准密度 → 测试: 高密度
       挑战: 冠层结构、纹理特征差异

场景2: 跨水肥管理泛化
       训练: 标准管理 → 测试: 控肥管理
       挑战: 叶色、长势差异

场景3: 跨时相泛化
       训练: 晴天数据 → 测试: 阴天数据
       挑战: 光照、阴影变化
```
